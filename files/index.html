  • Creates internet gateway so that traffic can egress from nginx hosts
  • Configures route for egress traffic
  • Creates security group for nginx hosts
  • Creates security group for alb load balancer
  • Creates load balancer, with target_group and http and https listeners
  • Creates IAM role, role_policy, and instance profile
  • Creates autoscaling group which launches latest canonical ubuntu with IAM profile to access private S3 bucket, installs nginx, copies this file to nginx root directory using user_data
  • Outputs the ELB dns name
